# MSc Thesis Presentation

**Multi-Omics Characterization of Abdominal Aortic Aneurysm via 3D Image-Derived Phenotypes**

## View online

https://bioinumer.github.io/MSc-Thesis-Presentation/

Open the link in any browser to present — no install needed.

## Local use

Open `presentation.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/presentation.html
